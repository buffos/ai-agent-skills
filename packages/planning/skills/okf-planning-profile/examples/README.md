# OKF Planning Profile Example

This example demonstrates the planning profile in a small but representative
slice.

It shows:

- one root project node
- one top-level capability
- one child capability
- one shared concern
- outward links to PRD, issue, and ADR artifacts

The files are examples of node shape and relationships. They are not meant to
be copied verbatim into a real project without adapting paths, titles, and
states.
